## 1.功能模块

### 1.1 佩戴装备

该功能会自动查询背包中最合适的装备信息，如果没有佩戴进行佩戴，筛选方式为同类型装备下优先考虑品质和等级，如果不止一个装备可对其进行优先级指定排序。





### 1.2 强化装备

该功能主要是对已经佩戴的装备进行强化，目前对设置的最高强化等级为7（也是安全等级上限，再往上面强化就有可能损坏），在强化的时候会考虑是否需要使用发光的强化卷轴，以及考虑强化卷轴可以交易的情况是否还会使用。

如果存在不可以交易的发光强化卷轴会优先使用，之后才会考虑不可交易普通强化卷轴

**注意：**目前设置的强化装备的起始品质是蓝色，也就是说蓝装以下的装备不会进行强化，如需修改可联系



### 1.3 人物造型

该功能会筛选一个最适合的人物造型进行使用，同品质下可以根据实际需求控制优先级。





### 1.4 使用宠物

该功能会筛选一个最合适的护卫者进行使用，同品质下可以根据实际需求控制护卫者的优先级。

## 2.脚本模块运行流程

目前脚本中主要穿插了以下模块

- **登录模块**

- **主线模块**
- **支线模块**
- **副本模块（三个周本，四个天本）**
- **挂机模块**
- **中间穿插的各种功能**

正常情况下（各个模块开关都开启），会先进行主线的推进，如果在**主线中死亡超过两次会先去挂机两小时**，在期间如果满足支线的运行条件也会进行支线推进，当满足副本的进入条件之后也会进行副本，这几个模块之间会有一个随机性，但在天副本和周副本之间会优先进行天副本，期间会穿插着各种功能的运行，如果当前已经没有了可以继续的副本以及主线支线的推进，就会进行挂机。

## 3.挂机模块

- 挂机功能主要是针对野外挂机，对于挂机的坐标点需要自己进行设置

- 挂机**相关的配置开关请在配置管理器中的配置说明进行查看**
- 脚本运行中，其他模块开关都打开并且满足条件的情况下，会先进行其他模块的运行（比如主线，支线，天副本，周副本...），然后才会进行挂机的运行，**挂机默认的是挂机两个小时**
- **在主线进行中如果死亡超过两次，也会先进行挂机两个小时**



### 3.1 挂机坐标点位置

当脚本运行之后，在Script同级目录下会看到一个**坐标设置.ini**，点击打开查看时会看到**[野外挂机]**下已经写入一个默认的挂机坐标点：

```ini
[野外挂机]
1=MID：14,X：125,Y：54,Z：67,NLV：1,MLV：100,R：40,M：3,L：1,MOD：0,PK模式：0,AREA：0,TIME：3600
```

可以参考该模版进行挂机坐标的录入，关于其中的配置如下：

| 名称 | 介绍                       |
| ---- | -------------------------- |
| MID  | 地图的id                   |
| X    | x坐标                      |
| Y    | y坐标                      |
| Z    | z坐标                      |
| NLV  | 该坐标点可以挂机的最小等级 |
| MLV  | 该坐标点可以挂机的最大等级 |

后面没有介绍的现在先不用管，后续如果运用了会再做讲解



**切记：**⚠⚠⚠⚠

- **等号左边依次填入连续数字作为序号，中途不可中断，如1,2,4，会影响脚本解析**
- **该坐标添加只能放在【野外挂机】下，不可放其他区块下**
- **该ini由于是在每台机器下，所以是只对该台机器起作用，不要将其与配置管理器中的配置混淆**



### 3.2 挂机坐标点设置

上面介绍了ini中的记录的坐标点是什么意思，接下来讲解怎么录制坐标点

1. 游戏中将角色移动到需求的坐标点
2. 停止移动，点击F2
3. 此时在调试台可以打印出如下数据

```ini
---------输出ini内置坐标-------------
idx=MID：1214,X：115,Y：50,Z：58,NLV：1,MLV：100,R：20,M：3,L：1,MOD：0,PK模式：0,AREA：0,TIME：3600
```

4. 将idx开头那行的数据双击，复制下来
5. 打开**坐标设置.ini**，将其复制在**[野外挂机]**最底下，修改idx为正确的序号，修改想自定义的数据（如最大最小等级）



### 3.3 挂机坐标点实际运用

当脚本要进行挂机后，会根据该ini中记录的坐标点进行筛选，其中会根据该台机器下挂机的角色数量平均坐标点的分配，

例如：ini中记录了3条坐标点，该台机器下第一个进行挂机的角色会选取第1个坐标点，而第2角色会选取第二坐标点，以此类推，第四个角色会又会继续选取第1个坐标点，依此循环选取坐标点，达到一个分配均匀的作用
